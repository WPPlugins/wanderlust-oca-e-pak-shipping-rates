Obtain OCA shipping rates dynamically via the OCA E-Pak API for your orders.

== Description ==

Obtenga tarifas de envío con OCA dinámicamente a través de la API OCA E-Pak para sus pedidos.

Con este complemento podrá obtener automáticamente las tarifas de envío de OCA en la página del carrito y la checkout con WooCommerce.

== Installation ==

1. Install the WooCommerce OCA Shipping - Plugin 
2. Activate the plugin
3. Go to the WooCommerce Settings Page 
4. Access Shipping Tab
5. Select "OCA E-Pak"
6. Check the Enable/Disable Checkbox.
7. Enter the settings that you would like you to use
8. Enter the dimensions of the products, (OCA uses this to determine suitability of products)

---------------------------------------

1) Instala y activa el plugin WooCommerce OCA Shipping en tu tienda.

2) Ingresá a la opción “Ajustes” del menú Woocommerce, en el panel izquierdo de tu WordPress.

3) Acede a la pestaña “Envio”

4) En el sub menú “Zonas de envio” selecciona “Oca Express Pack”

5) Hace clic en “Activar este método de envio”

6) Completa los datos necesarios: el codigo postal de origen, tu CUIT y los Boxsizes con sus respectivas operativas (estos datos se obtienen de OCA).

7) No olvides asignarle las caracteristicas para el envío a tus productos: Peso y dimensiones.

 

Una vez configurado con tus datos, el plugin ya está listo para calcular los costos de envío en el carrito de tu tienda.
